# Passo: Pedro Página (Montagem Final & Publicação) 📄

Neste último passo, o agente **Pedro Página** organiza os componentes finais para a postagem no portal.

## Instruções:
1.  **Montagem:** Organize o título, o conteúdo markdown revisado, a imagem validada e os links internos.
2.  **Hierarquia:** Defina o layout final da notícia na página do Supabase.
3.  **Disparo:** Execute o `insert.js` para realizar a inserção final dos dados no banco de dados.
4.  **Confirmar Publicação:** Verifique se a notícia ficou online com a categoria e o autor corretos.
