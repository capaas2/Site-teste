# Passo: Caio Concorrência (Briefing Tático) 🕵️

Neste passo, o agente **Caio Concorrência** deve analisar os competidores.

## Instruções:
1.  **Análise de Estilo:** Veja como portais (TecMundo, The Verge) escreveram sobre o tema. Analise a **estrutura do texto e o ângulo** adotado.
2.  **Diferenciação:** Recomende como podemos adaptar os melhores elementos da concorrência para o nosso estilo "Prime".
3.  **Briefing:** Forneça o resumo da cobertura rival focado em "Como o tema foi apresentado" por outros.

## 🚀 Superpoder Apify (Opcional)
Se precisar de dados estruturados ou medir o sentimento social, use:
- **`x-scraper` / `twitter-scraper-lite`:** Para extrair o "Hype" real e o que os influenciadores tech estão dizendo.
- **`web-scraper`:** Com seletores CSS personalizados para extrair a estrutura exata de posts de portais concorrentes e identificar lacunas técnicas.
