# 🗣️ Tom de Voz: FolhaByte

> Referência Kronos: `regras_folhabyte/estilo_de_escrita.md`

## 🎯 Posicionamento
O portal FolhaByte se posiciona como uma **autoridade técnica com "pés no chão"**. Não somos apenas um agregador de notícias; somos analistas.

## Princípios de Voz

| Princípio | O que fazer | O que NÃO fazer |
|-----------|-------------|-----------------|
| **Autoridade Acessível** | Use termos técnicos + explique o benefício real | Jargões sem explicação |
| **Direto ao Ponto** | Cada frase carrega informação ou análise | Enchimento, parágrafos vazios |
| **Escrita Visual** | Descreva hardware como se o leitor pudesse tocar | Textos 100% abstratos |
| **Ceticismo Saudável** | Questione preço, utilidade e concorrência | Aceitar press release como verdade |
| **Voz Ativa** | Prefira voz ativa para dinamismo | Voz passiva excessiva |
| **Frases Curtas** | Até 15 palavras no máximo por frase | Parágrafos-rio de 5 linhas |

## 🚫 Proibições de Redação
1. **Conclusões de IA**: Banido "Conclusão", "Para finalizar", "Em resumo". Use **"Análise de Impacto"**.
2. **Listas Numeradas Narrativas**: Reserve listas apenas para specs técnicas. Use texto fluido.
3. **Jargões Intraduzíveis**: O leitor precisa entender sem ir ao Google.
4. **Burying the Lede**: Fato central na **primeira linha**.
5. **Falso Hype**: Sem superlativos vazios para lançamentos padrão.

## ✅ Vocabulário

### Use
- Inovação, impacto, análise, desempenho, recursos, funcionalidade, inteligência artificial, oficial.

### Evite
- Supostamente, revolucionário, chocante, impressionante (exceto review emotivo), clique aqui.
- "Imperdível", "Confira agora", "Você precisa ver", "Bom dia leitor".
